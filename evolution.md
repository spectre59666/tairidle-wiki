# Evolution Methods

This page lists all the special evolution methods that some pokemon requires such as stones, held items, locations, moves and special methods.




**Methods:**
- **Stone Evolutions** — This is the list of each pokemons that evolves by using the *evolution stones* and link cable.
- **Held Items Evolutions** — This is the list of each pokemons that evolves by giving them a *specific item*. 
    - **Buyable Held Items** — This is the list of every *Held Items* that you can buy, and where to buy them.
    - **Wild Drop Held Items** — This is the list of every *Held Items* that you can obtain by catching specific wild Pokémon that has 1/20 of holding the item.
- **Location Evolutions** — This is the list of every *Locations* that you need to be in to evolve specific Pokémons.
- **Known Move Evolutions** — This is the list of every *Moves* certain Pokémons needs to know to evolve.
- **Party Member Evolutions** — This is the list of every pokemons that evolves by having a specific *Party Member*. 
- **Special Evolution Methods** — This is the Pokémons that requieres a different or atypical evolution method than any other Pokémons.
